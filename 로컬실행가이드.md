# 네이버 플레이스 크롤링 앱 - 로컬 실행 가이드 💻

## 📋 시스템 요구사항
- Python 3.8 이상
- 2GB 이상 RAM
- 인터넷 연결

---

## 🚀 설치 방법

### 1단계: Python 설치 확인

**윈도우/맥/리눅스 공통**
```bash
python --version
```
또는
```bash
python3 --version
```

> 💡 Python이 없다면 https://python.org 에서 다운로드하세요.

---

### 2단계: 프로젝트 폴더 생성

**윈도우 (명령 프롬프트 또는 PowerShell)**
```cmd
cd Desktop
mkdir naver_place_crawler
cd naver_place_crawler
```

**맥/리눅스 (터미널)**
```bash
cd ~/Desktop
mkdir naver_place_crawler
cd naver_place_crawler
```

---

### 3단계: 필요한 파일 다운로드

이 폴더에 다음 파일들을 넣어주세요:
- `naver_map_crawler.py` (메인 앱)
- `naver_place_crawler_real.py` (실제 크롤링 모듈)
- `templates/index.html` (웹 인터페이스)
- `requirements.txt` (필요한 패키지 목록)

---

### 4단계: 패키지 설치

**윈도우**
```cmd
pip install flask playwright
```

**맥/리눅스**
```bash
pip3 install flask playwright
```

> ⏱️ 설치 시간: 약 1-2분

---

### 5단계: Chromium 브라우저 설치

**윈도우/맥/리눅스 공통**
```bash
playwright install chromium
```

> ⏱️ 설치 시간: 약 1-2분 (약 100MB 다운로드)

---

### 6단계: 앱 실행

**윈도우**
```cmd

```

**맥/리눅스**
```bash
python3 naver_map_crawler.py
```

다음과 같은 메시지가 나오면 성공:

## 🎯 사용 방법

1. **검색어 입력**: 예) "강남역 맛집", "홍대 카페", "하수구역빌"
2. **검색 버튼 클릭**
3. **결과 확인**:

4. **CSV로 저장**: 엑셀에서 바로 열기 가능

## ⚠️ 문제 해결

### Python을 찾을 수 없음
```
'python' is not recognized...
```
→ Python 설치 후 PATH에 추가하거나 `python3` 사용

### pip를 찾을 수 없음
```
'pip' is not recognized...
```
→ `pip3` 또는 `python -m pip` 사용

### 포트 충돌
```
Port 5004 is in use by another program
```
→ `naver_map_crawler.py` 파일에서 `port=5004`를 `port=5005`로 변경

### Chromium 실행 오류
```
libnspr4.so: cannot open shared object file
```
→ 리눅스의 경우 다음 명령어 실행:
```bash
sudo apt-get install libnss3 libnspr4 libatk1.0-0 libatk-bridge2.0-0 \
  libcups2 libdrm2 libxkbcommon0 libxcomposite1 libxdamage1 \
  libxfixes3 libxrandr2 libgbm1 libpango-1.0-0 libcairo2 libasound2
```
