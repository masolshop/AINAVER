# 네이버 지도 크롤링 앱 🗺️

코딩 몰라도 사용할 수 있는 네이버 지도 크롤링 도구입니다!

## 🌟 주요 기능

- ✅ **쉬운 웹 인터페이스**: 클릭만으로 간편하게 사용
- ✅ **자동 데이터 수집**: 장소명, 주소, 전화번호, 평점, 리뷰 수 자동 수집
- ✅ **엑셀 저장**: CSV 파일로 저장하여 엑셀에서 바로 열기 가능
- ✅ **맞춤 검색**: 원하는 키워드로 검색 가능

## 🚀 시작하기

### 1. 필요한 프로그램 설치

```bash
# Python 패키지 설치
pip install flask selenium webdriver-manager
```

### 2. 앱 실행

```bash
python naver_map_crawler.py
```

### 3. 웹 브라우저에서 접속

앱을 실행하면 나오는 주소로 접속하세요!

## 📖 사용 방법

1. **검색어 입력**: 찾고 싶은 장소 입력 (예: "강남역 맛집", "홍대 카페")
2. **결과 수 설정**: 가져올 최대 결과 개수 설정 (1~100개)
3. **검색 실행**: '검색' 버튼 클릭
4. **결과 확인**: 자동으로 수집된 정보 확인
5. **저장**: 'CSV로 저장' 버튼으로 파일 저장

## 💾 저장되는 정보

- 장소명
- 카테고리 (예: 카페, 음식점 등)
- 주소
- 전화번호
- 평점
- 리뷰 수

## ⚠️ 주의사항

- 네이버 지도 이용약관을 준수해주세요
- 과도한 크롤링은 IP 차단의 원인이 될 수 있습니다
- 수집된 데이터는 개인적 용도로만 사용해주세요

## 🛠️ 문제 해결

### Chrome 드라이버 오류

Chrome 브라우저가 설치되어 있어야 합니다.

### 검색 결과가 없음

- 검색어를 다르게 입력해보세요
- 네트워크 연결을 확인해주세요
