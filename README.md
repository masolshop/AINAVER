## 🌟 주요 기능

### 🔍 장소 검색 & 분석
- ✅ **쉬운 웹 인터페이스**: 클릭만으로 간편하게 사용
- ✅ **자동 데이터 수집**: 장소명, 주소, 전화번호, 평점, 리뷰 수 자동 수집
- ✅ **타지역업체 구분**: 주업체와 타지역업체를 명확히 구분

- 여러 키워드를 한 번에 검증

- 통계 대시보드 제공
- CSV로 일괄 저장 가능

## 🏪 타지역업체란?

네이버 플레이스에는 두 가지 유형의 업체가 있습니다:

| 구분 | 주업체 | 타지역업체 |
|------|--------|------------|
| **주소** | 상세주소 (도로명, 건물번호) | 동/구 단위만 표시 |
| **전화번호** | 일반 전화 (02, 031 등) | 070 가상번호 |
| **사진** | 매장 사진 있음 | 사진 없음 |
| **상호명** | 제한 없음 | 최대 30자 |
| **평점/리뷰** | 평점과 리뷰 표시 | 평점 없음 |
| **콘텐츠** | 상세 정보 풍부 | 최소 정보만 |

이 앱은 **타지역업체를 자동으로 감지하고 구분**하여 보여줍니다!

## 🚀 시작하기

### 옵션 1: 데모 모드 (온라인)

온라인 데모를 통해 앱이 어떻게 작동하는지 확인하세요!
- 시뮬레이션 데이터로 모든 기능 체험 가능
- 설치 불필요

### 옵션 2: 실제 크롤링 (로컬)

실제 네이버 지도 데이터를 크롤링하려면:

#### 1. 필요한 프로그램 설치

```bash
# Python 패키지 설치
pip install -r requirements.txt

# Playwright 브라우저 설치
playwright install chromium
```

#### 2. 앱 실행

```bash
python naver_map_crawler.py
```

#### 3. 웹 브라우저에서 접속

```
http://localhost:5003
```

## 📖 사용 방법

1. **검색어 입력**: 찾고 싶은 장소 입력 (예: "강남역 맛집", "홍대 카페")
2. **결과 수 설정**: 가져올 최대 결과 개수 설정 (1~100개)
3. **검색 실행**: '검색' 버튼 클릭
4. **결과 확인**: 자동으로 수집된 정보 확인
5. **저장**: 'CSV로 저장' 버튼으로 파일 저장

## 💾 저장되는 정보

- 장소명
- 카테고리 (예: 카페, 음식점 등)
- 주소
- 전화번호
- 평점
- 리뷰 수

## ⚙️ 기술 스택

- **Flask** - 웹 서버
- **Playwright** - 브라우저 자동화 (로컬 실행 시)
- **HTML/CSS/JavaScript** - 프론트엔드

## ⚠️ 주의사항

1. **합법적 사용**: 네이버 지도 이용약관을 준수해주세요
2. **적당한 사용**: 과도한 크롤링은 IP 차단의 원인이 될 수 있습니다
3. **개인 용도**: 수집된 데이터는 개인적 용도로만 사용해주세요
4. **크롬 필요**: 로컬 실행 시 Chromium 브라우저가 자동으로 설치됩니다

## 🛠️ 문제 해결

### 데모 모드에서 실제 데이터가 안 나와요

데모 모드는 시뮬레이션입니다! 실제 크롤링을 원하시면 로컬에서 실행해주세요.

### Chrome 드라이버 오류

```bash
playwright install chromium
```

위 명령어로 브라우저를 설치해주세요.

### 검색 결과가 없음

- 네트워크 연결을 확인해주세요
- 검색어를 다르게 입력해보세요
- 로컬 환경에서 실행 중인지 확인하세요
