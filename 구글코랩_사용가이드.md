## 🚀 실행 방법 (5분 완성)

### 1단계: 구글 코랩 접속

**방법 A: 파일 직접 업로드**
1. https://colab.research.google.com 접속
2. 좌측 상단 **"파일" → "노트 업로드"**
3. `네이버_플레이스_크롤링_코랩.ipynb` 파일 선택
4. 업로드 완료!

**방법 B: 구글 드라이브에서 열기**
1. `네이버_플레이스_크롤링_코랩.ipynb` 파일을 구글 드라이브에 업로드
2. 파일 더블클릭
3. 상단에 **"Google Colaboratory로 열기"** 클릭

## 🔧 문제 해결

### Q1: ngrok URL이 작동하지 않아요
**A**: 
- 셀 5️⃣이 실행 중인지 확인
- URL을 다시 복사해서 시도
- 코랩 세션이 종료되었을 수 있음 → 처음부터 다시

### Q2: 검색이 안 돼요
**A**:
- 서버 로그 확인 (셀 5️⃣ 아래)
- 네트워크 연결 확인
- 페이지 새로고침

### Q3: 2시간 후에 종료돼요
**A**:
- ngrok 무료 버전 제한
- https://ngrok.com 에서 가입 → 토큰 받기 → 셀 2️⃣에 입력

### Q4: 코랩이 너무 느려요
**A**:
- 구글 계정 확인 (로그인 필요)
- 다른 노트북 닫기
- GPU/TPU 런타임 필요 없음 (기본 CPU로 충분)
